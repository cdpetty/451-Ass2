p2.py runs on python2 using the pydasm library found at https://github.com/axcheron/pydasm

The input hexstring is read from the file "example.hex" included in teh same directory as p2.py
We have included an example "example.hex" file which inbcludes the default hex string from https://www.onlinedisassembler.com/odaweb/

To run, simply use
$python p2.py
